# Rejap.ai Frontend

The frontend of Rejap.ai is a modern, responsive React application built with Vite and TypeScript. It provides an interactive and personalized Japanese learning experience.

## Tech Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4.1](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix UI)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router DOM](https://reactrouter.com/)

## Project Structure

- `src/components/`: Reusable UI components (Navbar, Sidebar, NextStepsCard, etc.).
- `src/pages/`: Main application pages (Home, LearningPath, Onboarding, Learning, Quiz).
- `src/lib/`: Utility functions and hooks.
- `src/App.tsx`: Main routing and application structure.
